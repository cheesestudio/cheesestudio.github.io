# Radmin VPN 联机使用指南

## ✅ 支持人数
最大支持 1 主机 + 3 客户端 = 总共 4 名玩家同时联机

---

## 🛠️ 联机步骤

### 主机操作
1. 打开 Radmin VPN，创建或加入一个网络
2. 运行中继程序：`node relay.js`
3. 打开 `bullet-hell.html`，点击菜单中的 **创建房间**
4. 将你的 Radmin VPN IP 地址告诉其他玩家

### 客户端操作
1. 打开 Radmin VPN，加入主机所在的网络
2. 打开 `bullet-hell.html`，点击菜单中的 **加入游戏**
3. 输入主机的 Radmin IP 地址，点击连接
4. 等待所有玩家准备完成后，主机点击开始游戏

---

## 🔌 端口信息
- TCP 端口: `7` (Radmin 标准端口，不需要映射)
- WebSocket 端口: `8007` (本地使用，不需要对外开放)

---

## ✔️ 网络要求
✅ Radmin VPN 已经正确连接
✅ 所有玩家在同一个 Radmin 网络
✅ Windows 防火墙允许端口 7 通讯
✅ 不需要公网 IP，不需要端口映射，不需要路由器设置
